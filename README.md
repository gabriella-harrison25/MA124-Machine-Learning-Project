# MA124-Machine-Learning-Project
Compact code sample: Python data manipulation, visualization, and modelling - numpy, scikit-learn, matplotlib, pandas

This repository contains an excerpt from a year-long Machine Learning project in Python focused on data visualization, manipulation, and modelling. It includes a Scikit-learn pipeline for the *SeoulBikeData_mod.csv* dataset (Task A1) and a from-scratch numpy gradient descent algorithm (Task A2) on a toy dataset.
